# UnyieldingChinese

# 一生要强的中国人模拟器

## 完整产品设计方案

### 一、产品核心定位与底层逻辑

#### 1. 核心定位

国内首个基于**「人生时间窗口双维度代价模型」**的全生命周期决策辅助与规划工具。
拒绝制造焦虑，核心价值是帮用户看清「提前透支」和「滞后错过」两大人生陷阱，实现人生时间、精力、金钱资源的最优配置，在对的时间做对的事，过不后悔的人生。

- Slogan：**人生没有标准答案，但有最优时机**
- 核心受众：16-45 岁处于人生关键决策期的用户，覆盖学生、职场新人、成家立业人群、中年转型人群
- 产品形态：响应式 Web 网站（适配手机/平板/PC），后续可延伸小程序/APP

#### 2. 底层核心理论模型

人生不是线性衰减，而是**分阶段的窗口式机会**。每个核心人生事件，都存在「绝对禁忌提前期、提前高代价期、黄金执行期、滞后高代价期、绝对不可逆关闭期」5 个时间区间，且同时存在「提前透支的代价」和「滞后错过的代价」，不存在「越早越好」或「越晚越稳」的绝对选项。

#### 3. 单窗口标准化数据模型（全系统统一规范）

每个窗口卡片均遵循以下标准化字段，确保系统可量化、可追溯、有公信力：
| 字段分类 | 核心内容 |
|----------|----------|
| 基础属性 | 窗口唯一 ID、窗口名称、所属人生维度、一句话核心定义、底层科学/行业依据来源 |
| 时间轴属性 | 绝对禁忌提前期、提前高代价期、黄金执行期、滞后高代价期、绝对不可逆关闭期 |
| 量化评估指标 | 锁死强制力（0-100%，窗口关闭后不可逆的程度）、提前代价等级、滞后代价等级、黄金期 ROI、滞后期 ROI |
| 适配规则 | 性别/职业/城市/人生阶段的适配标签，差异化调整时间区间与代价等级 |
| 落地内容 | 黄金期推荐行动、提前期避坑指南、滞后期补救方案、核心风险提示 |

---

### 二、系统整体架构

全系统分为 6 大核心模块，形成「用户初始化-全景洞察-风险评估-行动落地-长期追踪」的完整闭环：

```
用户画像初始化系统 → 核心窗口库引擎 → 动态仪表盘与可视化系统 → 双维度代价评估与预警系统 → 人生规划与行动落地系统 → 人生模拟与长期追踪系统
```

---

### 三、核心模块详细设计

#### 模块一：用户画像与初始化系统

用户进入网站的第一步，通过极简流程完成个性化配置，避免全量卡片的信息过载，生成专属仪表盘。

1. 基础信息采集（必填，30 秒完成）
   - 核心字段：当前年龄、性别、所在城市、最高学历、职业状态、婚姻状态、子女情况
   - 自动匹配：根据年龄自动划分人生阶段（学生期/职场新人期/成家立业期/中年稳定期/退休预备期）
2. 人生优先级配置（选填，10 秒完成）
   - 提供 6 大人生维度，用户选择 3 个最高优先级，系统自动高亮对应维度的窗口，排序前置
   - 6 大维度：生理与健康、学业与职业、财富与资产、亲密与家庭、精神与成长、风险与兜底
3. 风险偏好配置（选填，5 秒完成）
   - 保守型/平衡型/进取型，系统差异化调整行动方案的激进程度（如保守型优先推荐兜底保障，进取型优先推荐成长型机会）
4. 账号体系
   - 支持游客模式（体验核心仪表盘功能）、手机号注册（保存配置、规划、打卡记录，多端同步）

#### 模块三：动态仪表盘与可视化系统（前端核心交互）

还原用户截图中的核心交互，并做全面升级，是用户进入网站后的主页面，核心是「年龄滑块+动态卡片+全景可视化」。

1. 核心交互：年龄时间轴滑块
   - 顶部设置 0-80 岁可拖动的年龄滑块，用户拖动到对应年龄，系统实时刷新所有窗口的状态、颜色、排序，实现「滑动看人生全貌」的核心体验
   - 滑块默认定位到用户初始化的当前年龄，支持点击跳转、拖动滑动，配有顺滑的动画效果
2. 窗口状态视觉规范（颜色统一，一眼识别）
   | 状态标签 | 颜色 | 含义 | 排序优先级 |
   |----------|------|------|------------|
   | 黄金执行期 | 低饱和青绿色 | 当前处于最优区间，优先行动 | 1（最前置） |
   | 即将关闭预警 | 橙色 | 1-3 年内将进入绝对关闭期，需抓紧补救 | 2 |
   | 提前高风险 | 黄色 | 当前处于提前高代价期，需避坑，不要盲目行动 | 3 |
   | 未到开启期 | 浅灰色 | 还未到窗口开启时间，可提前了解 | 4 |
   | 已关闭/不可逆 | 浅红色 | 窗口已进入绝对关闭期，无法补救 | 5（最后置） |
3. 页面布局
   - 顶部：人生阶段概览（当前年龄、所处人生阶段、核心优先级提示、即将关闭窗口预警条数）
   - 筛选栏：支持按 6 大维度、窗口状态、优先级筛选卡片，支持一键切换「仅看我的黄金窗口」「仅看预警窗口」
   - 主区域：响应式卡片网格，PC 端 5 列、平板 3 列、手机 2 列，每个卡片显示：状态标签、窗口名称、核心一句话提示、锁死强制力、代价等级；点击卡片弹出全屏详情页，展示完整时间轴、科学依据、行动指南、避坑提示
   - 侧边栏（PC）/底部栏（手机）：核心数据看板，包含「当前黄金窗口数量」「即将关闭预警窗口」「高风险提前窗口」「人生完成度评分」「人生维度雷达图入口」
4. 全景可视化模块
   - 人生全景时间轴：0-80 岁完整时间轴，标注所有窗口的开启、黄金期、关闭节点，用户可滑动查看人生全貌，一眼看清哪些窗口已过、哪些正在进行、哪些未来到来
   - 人生维度雷达图：6 大维度的完成度评分，直观展示用户的优势维度与短板维度
   - 代价曲线可视化：每个窗口详情页内置「提前/滞后代价曲线」，横轴为年龄，纵轴为代价/风险，直观展示黄金期的成本最低点，以及提前/滞后的代价上升趋势

#### 模块四：双维度代价评估与风险预警系统

这是本系统和市面上所有焦虑型工具的核心差异化优势，同时量化「提前的代价」和「滞后的代价」，帮用户避开两大人生陷阱。

1. 双维度代价矩阵
   每个窗口详情页内置 2x2 代价矩阵，横轴为「提前程度」，纵轴为「滞后程度」，四个象限直观展示风险：
   - 最优象限：黄金执行期，低提前风险+低滞后风险，ROI 最高
   - 风险象限 1：过度提前，高提前代价，易造成不可逆损伤
   - 风险象限 2：过度滞后，高滞后代价，补救成本指数级上升
   - 最差象限：既提前透支，又错过补救期，终身不可逆
2. 智能预警系统
   - 关闭预警：自动识别未来 1-3 年内即将关闭的高锁死强制力（≥80%）窗口，在首页顶部置顶预警，标注「剩余时间」「滞后代价」「补救方案」
   - 提前风险预警：自动识别用户当前年龄处于提前高代价期的窗口，在卡片上标注黄色预警，提示「当前行动过早，需避坑」，并给出提前风险的具体影响
   - 不可逆风险提示：对锁死强制力 ≥90%的窗口，全程标红重点提示，明确告知不可逆风险
3. 个性化代价计算器
   支持用户自定义决策模拟，比如「我现在 28 岁，想 35 岁再生育，需要付出多少代价」，系统自动量化输出：生理风险提升比例、辅助生殖成本提升幅度、精力消耗预估、机会成本损失，让用户直观看到决策的长期影响。

#### 模块五：人生规划与行动落地系统

拒绝只制造焦虑，核心是帮用户落地行动，形成「看全景-做决策-拆目标-追进度」的完整闭环。

1. 个性化人生规划生成器
   用户点击「生成我的人生规划」，系统基于用户画像、优先级、当前年龄，自动生成「1 年行动清单、3 年规划、5 年全景」，按优先级分为 4 类：
   - 必做项：当前黄金期的高优先级窗口，必须投入核心资源
   - 避坑项：当前提前高代价期的窗口，严禁盲目行动
   - 预备项：未来 1-3 年即将开启的窗口，提前做准备
   - 补救项：滞后高代价期的窗口，仍有补救机会，制定最低成本补救方案
2. 目标拆解与打卡系统
   每个行动项支持拆解为可执行的最小单元，比如「考研上岸」拆解为「每月复习计划-每周学习目标-每日打卡任务」，用户可完成打卡，系统自动追踪进度，更新人生完成度评分。
3. 场景化专属解决方案
   针对不同人生阶段的用户，提供开箱即用的专属方案，比如：
   - 18 岁高考毕业生：大学 4 年全景规划（学历、技能、人脉、财商启蒙）
   - 25 岁职场新人：3 年职业成长+第一桶金积累方案
   - 30 岁成家人群：家庭规划+子女教育+资产配置+父母养老一体化方案
   - 40 岁中年人群：职业护城河+被动收入+健康管理+风险隔离方案
4. 里程碑管理系统
   用户完成核心人生目标（如考研上岸、结婚、买第一套房、被动收入达标），可记录为人生里程碑，系统自动更新人生全景图，生成专属的人生里程碑报告。

#### 模块六：人生模拟与长期追踪系统（杀手锏功能）

1. 人生选择模拟器
   支持用户模拟不同人生选择的长期影响，比如：
   - 模拟 1：22 岁毕业就买房 vs 30 岁再买房，对职业发展、财富积累、创业机会的全周期影响
   - 模拟 2：30 岁前拼事业 vs 30 岁前生娃，对人生各维度的长期影响
     系统基于窗口模型，量化输出两种选择的优劣势、代价差异、风险点，帮用户做出更理性的决策。
2. 年度人生复盘系统
   每年年底，系统自动生成用户的「年度人生复盘报告」，总结当年完成的里程碑、把握的黄金窗口、错过的机会、避坑的风险，并生成下一年的核心行动建议。
3. 人生档案系统
   完整记录用户的人生规划、打卡进度、里程碑、复盘报告，形成专属的人生数字档案，支持导出 PDF，永久保存。

### 四、UI/UX 设计规范

1. 设计风格：极简理性、温暖克制，拒绝焦虑感，主打「清晰、专业、有温度」
   - 主色调：低饱和青绿色（代表希望、理性、成长）
   - 辅助色：橙色（预警）、黄色（提示）、红色（风险）、灰色（中性）
   - 字体：无衬线字体，正文清晰易读，标题简洁有力
2. 卡片设计：圆角矩形，极简留白，信息层级清晰，无冗余元素
   - 卡片顶部：状态标签（带底色）
   - 卡片中部：窗口名称、核心一句话定义
   - 卡片底部：锁死强制力、代价等级，用进度条/图标直观展示
3. 响应式适配：优先适配手机端（抖音引流的核心用户场景），同时完美适配平板、PC 端，保证全设备体验一致
4. 交互原则：
   - 极简操作：核心功能 3 步内触达，游客无需注册即可体验核心仪表盘
   - 顺滑反馈：滑块拖动、卡片点击、弹窗弹出均配有柔和的动画反馈
   - 信息分层：核心信息一眼可见，详细信息点击展开，避免信息过载

### 八、产品核心价值观

1. 不制造焦虑，只提供清醒。我们不渲染「人生必须赢在起跑线」，只帮用户看清每个选择的代价，做出不后悔的决策。
2. 错过的窗口不必执念，当下的黄金期才是重点。对于已经关闭的窗口，我们教你接受；对于正在开启的窗口，我们教你把握。
3. 提前的透支，比错过更可怕。人生不是越早越好，稳扎稳打，在对的时间做对的事，才是最高效的人生。
