# 项目文件结构

## 📁 目录结构
```
UnyieldingChinese/
├── index.html              # 主页面（使用 Tailwind CSS + DaisyUI）
├── index.html.backup       # 原始文件备份
├── README.md              # 项目说明
├── REFACTORING_SUMMARY.md # 重构总结
├── PROJECT_STRUCTURE.md   # 本文件
└── assets/
    ├── css/
    │   └── styles.css    # 自定义样式（动画、效果等）
    └── js/
        ├── data.js        # 完整窗口数据（40个）
        └── app.js         # 主应用逻辑
```

## 🚀 快速开始

### 方法 1：直接打开
直接在浏览器中打开 `index.html` 文件即可使用。

### 方法 2：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问：`http://localhost:8000`

## 📊 数据说明

### 窗口数据（data.js）
- 包含 40 个完整的人生时间窗口数据
- 覆盖 6 大维度：生理健康、学业职业、财富资产、亲密家庭、精神成长、风险兜底
- 每个窗口包含：时间范围、锁死强制力、代价评估、行动建议、科学依据

### 应用逻辑（app.js）
- 数据加载：使用动态 script 标签加载 windowData
- 核心功能：
  - 窗口状态计算
  - 维度雷达图渲染
  - 全景时间轴渲染
  - 卡片网格渲染（支持维度和状态筛选）
  - 详情弹窗（标签页导航）
  - 年龄滑块交互
  - 快捷跳转功能

## 🎨 技术栈

- **Tailwind CSS** - 实用优先 CSS 框架
- **DaisyUI** - 现代化组件库
- **Google Fonts (Inter)** - 专业字体
- **Vanilla JavaScript** - 无框架依赖

## 🔄 数据流

```
index.html
    ↓
加载 styles.css
    ↓
加载 app.js
    ↓
加载 data.js
    ↓
初始化应用
    ↓
渲染界面
```

## 📱 响应式支持

- **移动端** (320px - 480px)：单列卡片
- **平板端** (481px - 768px)：双列卡片
- **桌面端** (769px - 1200px)：三列卡片
- **大屏端** (1200px+)：四列卡片

## 🎯 核心功能

1. **用户信息区** - 渐变色背景、人生阶段识别、4大指标统计
2. **维度雷达图** - 6 大维度完成度可视化、动态柱状图
3. **年龄滑块** - 快捷跳转按钮、实时更新所有视图
4. **全景时间轴** - 可视化所有窗口、点击交互、游标定位
5. **双重筛选** - 维度筛选 + 状态筛选（黄金期/预警/提前/已关闭）
6. **卡片网格** - 现代化设计、悬停效果、一键详情
7. **详情弹窗** - 标签页导航（概览/详情/行动）、分层信息展示

## 🔧 维护说明

### 修改数据
编辑 `assets/js/data.js` 文件，修改或添加窗口数据。

### 修改样式
编辑 `assets/css/styles.css` 文件，自定义动画和效果。

### 修改逻辑
编辑 `assets/js/app.js` 文件，修改应用逻辑。

### 重置样式
如需重置为纯 Tailwind CSS，删除或清空 `assets/css/styles.css`。

## 📝 开发建议

1. **数据管理**：所有窗口数据集中管理，避免重复
2. **组件复用**：使用 DaisyUI 组件保持一致性
3. **性能优化**：事件委托、合理使用 CSS 动画
4. **代码规范**：统一缩进、命名规范、注释清晰
