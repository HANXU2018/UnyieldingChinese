# 使用说明

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可使用：

## 📊 核心功能

### 1. 用户信息区

- **渐变色顶部栏** - 视觉冲击力强
- **4 大核心指标** - 黄金期、即将关闭、提前风险、已关闭
- **人生阶段识别** - 自动根据年龄显示（幼儿期 → 退休期）
- **毛玻璃效果** - 现代化的视觉质感

### 2. 维度雷达图

- **6 大维度可视化** - 生理健康、学业职业、财富资产、亲密家庭、精神成长、风险兜底
- **动态柱状图** - 实时显示各维度完成度百分比
- **图标标识** - 每个维度都有专属图标

### 3. 年龄滑块

- **0-80 岁范围调节** - 实时更新所有视图
- **7 个快捷跳转** - 0/18/25/30/40/50/60 岁，一键跳转
- **大字体显示** - 当前年龄突出显示

### 4. 全景时间轴

- **40 个窗口可视化** - 时间轴上的彩色窗口块
- **颜色编码** - 黄金期（天蓝）、即将关闭（橙色）、提前风险（黄色）
- **交互性** - 点击时间轴窗口可直接打开详情
- **游标标记** - 当前年龄位置，拖拽滑块实时移动

### 5. 双重筛选

- **维度筛选** - 6 大维度（生理健康、学业职业、财富资产、亲密家庭、精神成长、风险兜底）
- **状态筛选** - 仅黄金期、预警、提前、已关闭
- **组合使用** - 可同时筛选维度和状态

### 6. 卡片网格

- **响应式布局** - 移动端（1 列）、平板（2 列）、桌面（3-4 列）
- **卡片状态** - 边框颜色编码（黄金期/即将关闭/提前风险/已关闭）
- **核心信息** - 窗口名称、描述、锁死强制力、滞后代价
- **悬停效果** - 卡片抬起、阴影加深
- **点击交互** - 点击打开详情弹窗

### 7. 详情弹窗

- **标签页导航** - 概览/详情/行动三个标签页
- **分层信息展示** - 避免信息过载
- **时间轴可视化** - 开启/黄金期/关闭的时间范围
- **代价评估** - 提前代价、滞后代价的详细说明
- **行动建议** - 具体可执行的行动清单
- **科学依据** - 每个窗口的权威来源

## 🎨 设计特点

- **现代配色** - 天蓝色主色调，配合橙色、黄色、红色的状态色
- **毛玻璃效果** - 半透明背景 + 模糊效果
- **圆角设计** - 统一的圆角规范（8px/12px/16px）
- **阴影系统** - 柔和阴影，营造立体感
- **字体** - Inter 字体，清晰易读

## 📱 响应式支持

| 设备尺寸            | 列数 | 卡片大小 |
| ------------------- | ---- | -------- |
| 移动端 (≤480px)     | 1    | 单列     |
| 平板端 (481-768px)  | 2    | 双列     |
| 桌面端 (769-1200px) | 3    | 三列     |
| 大屏端 (>1200px)    | 4    | 四列     |

## 🔧 使用技巧

1. **快速浏览** - 使用快捷跳转按钮快速查看关键年龄段
2. **精准筛选** - 使用双重筛选快速定位关注内容
3. **拖动探索** - 拖动滑块在不同年龄查看窗口状态
4. **点击详情** - 点击卡片或时间轴窗口查看完整信息
5. **标签页切换** - 在详情弹窗中切换不同信息视角

## 📊 数据说明

- **40 个完整窗口** - 覆盖 0-80 岁全生命周期
- **6 大维度** - 每个维度包含多个相关窗口
- **科学依据** - 每个窗口都标注权威来源
- **行动建议** - 针对当前状态的具体可执行建议

## 🐛 故障排除

### 页面无法显示

- 检查浏览器控制台是否有 JavaScript 错误
- 确认所有文件路径正确

### 样式异常

- 检查 assets/css/styles.css 是否存在
- 清除浏览器缓存后刷新

### 数据不显示

- 确认 assets/js/data.js 文件存在
- 打开开发者工具检查网络请求
